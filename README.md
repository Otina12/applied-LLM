# LLM Systems

This repository contains a collection of tasks and projects focused on large language model systems, applied machine learning, and agent based workflows.

## Repository structure

### Tasks
Each folder contains an individual task.  
Every task includes its own description in the notebook explaining the goal, input, and expected outcome.

### Projects
The projects folder contains four larger projects.  
Each project has its own README.md file with a detailed explanation.

1. [01-plagiarism-detection](projects/01-plagiarism-detection/README.md)  
   Plagiarism detection using embeddings, BM25, FAISS, and model based evaluation.

2. [02-nosql-agent](projects/02-nosql-agent/README.md)  
   A simple HR assistant that manages employees stored in a SQLite database.

3. [03-multi-agent-ml](projects/03-multi-agent-ml/README.md)  
   Multi-agent ML team assisstant (Data Cleaning, Data Engineering and Model Trainer agents).

4. [04-resume-optimizer](projects/04-resume-optimizer/README.md)  
   An end to end system that analyzes resumes and rewrites sections to better match job descriptions.

## Notes
This repository is intended for learning, experimentation, and reference.  
All examples and projects are educational and self contained.
